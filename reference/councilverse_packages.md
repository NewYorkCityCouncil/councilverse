# List all packages imported by councilverse

List all packages imported by councilverse

## Usage

``` r
councilverse_packages()
```

## Examples

``` r
councilverse_packages()
#> [1] "cli"          "councildown"  "councilcount" "crayon"       "rstudioapi"  
#> [6] "utils"        "htmltools"    "tidyverse"   
```
